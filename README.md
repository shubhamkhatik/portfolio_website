## ğŸ Getting Started

### Prerequisites

- **Node.js** 18+ 
- **npm**, **yarn**, **pnpm**, or **bun**

### Installation

```bash
# Clone the repository
git clone https://github.com/shubhamkhatik/Portfolio_Website.git
cd Portfolio_Website

# Install dependencies
npm install

# Start the development server
npm run dev
```
